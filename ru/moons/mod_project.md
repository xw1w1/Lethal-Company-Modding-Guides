# Создание мода
Если вы уже импортировали все нужные зависимости в проект, можно приступить к созданию самого мода.

Поскольку для планеты используется `LethalLevelLoader`, для начала мы сделаем сам мод LLL.

В папке `Assets/LethalCompany/Mods` создайте свою папку рядом с `plugins`.

<img width="415" height="232" alt="image" src="https://github.com/user-attachments/assets/65bc403b-3011-4c71-8e08-b89a6f2070ea" />

Внутри этой папки создайте `ExtendedMod`. (`ПКМ > LethalLevelLoader > ExtendedMod`)

В вашей папке должен появиться следующий `ScriptableObject`:
<img width="854" height="367" alt="image" src="https://github.com/user-attachments/assets/faca4f33-86de-4df5-855e-248278e95bf8" />

Поле `Mod Name`, как можно догадаться, отвечает за название вашего мода.
Поле `Author Name` - имя автора. В моем случае это `Turbotaliz`. Учтите, что лучше использовать одинаковое имя автора для всех ваших проектов :')

Поля с **Extended** контентом, это то, что мы хотим добавить в наш мод в будущем.

### Структура проекта
Я добавлю эту сноску просто для общего понимания.

В будущем мы разобьем проект на бандлы, один для сцены, и только сцены, (Scene Asset), второй для всех нужных файлов мода (ExtendedMod, ExtendedLevel, SelectableLevel и т.д).

Структура проекта может отличаться, но я придерживаюсь следующей схемы:

```
LethalCompany/
\Game
\Mods
-\plugins [Файлы для BepInEx]
-\TurbotalizsMoons [Корневая папка в которой я храню свои проекты]
--\12Aerona [Сам мод Aerona, одна из моих планет]
---\Include [Музыка и некоторые меши, которые я использую в сцене]
---\ExtendedLevel, ExtendedMod и т.д. [Сюда же можно поместить и сцену, на самом деле]
--\SharedPrefabs [Префабы которые я заранее сделал. Полки, ангары, готовые деревья и прочий декор, который я утащил с других карт. Он не входит в сам мод, только в его сцену и может быть использован в других модах, по этому они и Shared.]
--\BlankReferences [До этого мы дойдем позже в разделе оптимизаций]
--\Scenes [Папка для ассетов сцен, но это только по желанию]
```
